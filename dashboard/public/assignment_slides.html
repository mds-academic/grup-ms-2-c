Kondisi Tunggal vs Ganda 🥊

1. If-Then (Tunggal):

  • Jika lapar, maka makan. (Kalau tidak lapar? Tidak ada instruksi).

2. If-Then-Else (Ganda):

  • Jika lapar, maka makan. Selain itu (Else), minum air.

Else memberikan jawaban alternatif saat kondisi TIDAK terpenuhi.
