Mini Project: Safe Transaction Input 🛡️

Sekarang, saatnya kamu membuat program pencatatan transaksi yang aman dan anti-kotor!

Aturan Program:

  1. Minta input nama transaksi (tolak jika kosong).
  2. Minta input jenis transaksi (tolak jika bukan "pemasukan" atau "pengeluaran").
  3. Minta input nominal transaksi (tolak jika <= 0).
  4. Tampilkan dictionary transaksi jika semua data valid.
