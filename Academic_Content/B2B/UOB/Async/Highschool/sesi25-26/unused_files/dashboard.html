Ringkasan Materi (Slide 1-4)

Video ini merangkum konsep dasar percabangan (Conditionals) dalam pemrograman:

  • Program yang Bisa Memilih: Program tidak hanya berjalan lurus. Ia bisa memilih jalan tergantung pada kondisinya.
  • Keputusan Sehari-hari: Sama seperti membawa payung jika hujan, komputer juga membuat keputusan berdasarkan kondisi tertentu.
  • Cara Komputer Berpikir: Jika syarat (kondisi) bernilai Benar (True), komputer akan menjalankan aksi tersebut.
  • Seperti Pintu Penjaga: Program mengecek apakah kondisinya terpenuhi (punya kunci). Jika ya, pintu perintah terbuka. Jika tidak, pintu tertutup!

Ringkasan Materi (Slide 5-12)

Di video ini, kita mempraktikkan langsung cara menyusun kode bersyarat di Python:

  • if Sederhana: Jika kondisi bernilai True, baris kode di bawahnya akan dijalankan. Jika False, akan dilewati.
  • Kondisi if-else: Bagaikan persimpangan jalan. Kalau kondisi if salah, maka otomatis bagian else yang akan jalan.
  • elif untuk Multi-Branch: Jika kita punya banyak opsi kategori (seperti rentang nilai A, B, C), gunakan elif untuk mengecek kondisi tambahan secara berurutan. Ingat, urutan kondisi di elif itu sangat penting karena program membacanya dari atas ke bawah!
